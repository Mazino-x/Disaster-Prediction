## Technology Stack

### Backend
- **Language:** Python 3.14
- **Framework:** Flask
- **ML:** scikit-learn
- **Data:** pandas, numpy
- **Deployment:** Gunicorn, WSGI

### Frontend
- **Framework:** React 18
- **Styling:** CSS3 (gradients, animations, blur)
- **Map:** Leaflet + react-leaflet
- **Build:** create-react-app

### Data
- **Source:** NOAA (historical) + Generated (synthetic)
- **Format:** CSV
- **Size:** 2,000+ clean records per model

## Deployment Instructions

### Quick Start
```bash
# Backend
python src/api/main.py

# Frontend
npm start

# Open browser
http://localhost:3000
```

### Production
```bash
# Build
npm run build

# Deploy (see DEPLOYMENT_CHECKLIST.md)
gunicorn --pythonpath src/api wsgi:app
```
